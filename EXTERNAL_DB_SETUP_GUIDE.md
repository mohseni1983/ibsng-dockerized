# External PostgreSQL Database Setup Guide

## Quick Start

To use an external PostgreSQL database, set these environment variables:

```bash
export IBSNG_DB_HOST="your-db-host-or-ip"
export IBSNG_DB_PORT="5432"
export IBSNG_DB_USER="ibs"
export IBSNG_DB_PASSWORD="your_password"
export IBSNG_DB_NAME="IBSng"
```

Then restart IBSng service.
